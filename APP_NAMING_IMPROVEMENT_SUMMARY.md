#### 포지셔닝별 이름 생성

**직관형 (intuitive)**
- 패턴: 키워드 + 도구/매니저 (예: "영어매니저", "수학헬퍼")
- 패턴: 의미 단어 + 앱 기능 (예: "연산노트", "로드맵가이드")

**감성형 (emotional)**
- 패턴: 성취/동기 + 키워드 (예: "성장영어", "도전수학")
- 패턴: 습관/일상 + 키워드 (예: "영어루틴", "수학일상")

**전문형 (professional)**
- 패턴: 키워드 + 전문성 (예: "영어프로", "수학마스터", "영어센터")
- 패턴: 키워드 + 체계 (예: "영어시스템")

**캐주얼형 (casual)**
- 패턴: 키워드 + 친근한 표현 (예: "영어친구", "수학파트너")
- 패턴: 의미 단어 + 캐주얼 표현 (예: "연산다이어리")
- 패턴: 키워드 합성 (예: "영어공부")

## C) 금지어/품질검사 규칙 코드

### 금지 패턴
```typescript
function isBadTitle(title: string): boolean {
  // 1. 1~2글자 차단
  if (t.length <= 2) return true;
  
  // 2. 자음만 있는 패턴 차단
  if (/^[ㄱ-ㅎ]+$/.test(t)) return true;
  
  // 3. 첫글자만 따서 만든 패턴 차단
  if (/^[가-힣]앱$|^더[가-힣]$|^[가-힣]프로?$/.test(t)) return true;
  
  return false;
}
```

### 금지 예시
- "영앱" (영어의 첫글자만)
- "더영" (더 + 첫글자)
- "영프로" (첫글자 + 프로)
- "ㅇㅇ앱" (자음만)
- "아앱" (1글자 + 앱)

### 허용 예시
- "영어매니저" (의미 있는 합성어)
- "성장영어" (의미 있는 합성어)
- "영어프로" (의미 있는 합성어, 2음절 이상)
